# 🎬 全网VIP视频免费解析脚本

## 📖 项目概述

这是一个强大的用户脚本，旨在帮助用户免费观看各大视频网站的VIP内容。通过集成多个第三方解析接口，该脚本能够绕过视频网站的会员限制，让用户享受免费的视频观看体验。

### 🌟 主要特性

- **多平台支持**：覆盖国内主流视频网站
- **智能解析**：集成多个解析接口，自动选择最优方案
- **自动播放**：支持自动解析和播放功能
- **响应式设计**：完美适配桌面端和移动端
- **可定制界面**：支持拖拽移动解析按钮位置
- **实时更新**：解析接口定期更新，确保可用性

### 功能特性说明

#### 1. 自动解析功能
- **默认状态**：关闭
- **工作原理**：页面加载后自动使用选中的解析接口进行解析

- **内嵌播放**：在原页面嵌入播放器

## ⚠ 常见问题与解决

### 1. 重音/音画不同步问题 (偶发)
- **可能原因：**
  - 网络连接速度过慢
  - 视频源服务器访问压力大
  - 解析接口负载过高
- **解决方法：**
  - 尝试刷新页面
  - 切换其他解析接口
  - 检查网络连接状态

### 2. 解析失败
- **可能原因：**
  - 视频链接格式不正确
  - 解析接口暂时失效
  - 网站更新了防护机制
- **解决方法：**
  - 确认视频链接是否完整
  - 尝试其他解析接口
  - 等待脚本更新

### 3. 按钮不显示
- **可能原因：**
  - 网络问题
  - 脚本未正确开启
  - 网站不在支持列表中
  - 浏览器插件冲突
- **解决方法：**
  - 更换网络或者反馈
  - 开启脚本，确保脚本运行
  - 检查网站是否在支持列表
  - 尝试禁用其他插件

### 4. 移动端使用问题
- **可能原因：**
  - 视频网站未设置为桌面端访问
  - 浏览器不支持用户脚本
  - 权限设置问题
- **解决方法：**
  - 观看视频时浏览器必须切换UA为桌面模式（电脑UA/查看桌面网站）
  - 使用支持用户脚本的浏览器（更换浏览器）
  - 检查脚本管理器权限设置

### 5. 自动解析不工作
- **可能原因：**
  - 自动解析功能未开启
  - 选择的接口失效
- **解决方法：**
  - 点击"关"按钮启用自动解析
  - 在VIP按钮中选择其他解析源
